# Debatní časovač (v3.1)

Moderní jednostránkový nástroj pro řízení debat v prohlížeči (časování řečníků, páry/role, náhodný výběr témat, vestavěný tutoriál). Vše běží offline – stačí otevřít soubor v prohlížeči.

## Obsah repozitáře
- [debatni-casovac-v3.1.html](debatni-casovac-v3.1.html) – kompletní aplikace s vloženými styly, skripty i ikonami.

## Jak spustit
1. Otevři [debatni-casovac-v3.1.html](debatni-casovac-v3.1.html) v libovolném moderním prohlížeči (funguje offline).
2. Použij obrazovku nastavení k zadání formátu debaty, dvojic a témat, poté spusť časovač.
3. Pro prezentaci nebo fullscreen přepni na obrazovku běhu a spusť časovač.

## Nasazení na GitHub Pages (doporučeno)
1. Na GitHubu vytvoř nový repozitář (např. `debatni-casovac`).
2. V kořenové složce spusť:
   ```bash
   git init
   git add debatni-casovac-v3.1.html README.md LICENSE .gitignore
   git commit -m "Add Debatni casovac v3.1"
   git branch -M main
   git remote add origin git@github.com:USERNAME/debatni-casovac.git
   git push -u origin main
   ```
3. V nastavení repozitáře zapni GitHub Pages (branch `main`, složka `/`).
4. Pokud nechceš veřejné šíření, nech repozitář jako private a sdílej pouze Pages URL s vybranými lidmi.
